3. Preferences

3.1. How to Change Your Password

  1. Log in and go to the top right hand corner of the screen. The name of the current user logged into TACTIC session is displayed here. Click on the button labelled "My Account". From the context menu, select "Edit My Account".

  2. Type in the new password and re-confirm it in the following field.

    Modify any other account information that needs to updated and hit Save.
